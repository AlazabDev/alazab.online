## 🚀 Quick Start (3 Steps)

### Step 1: Set API Key (2 minutes)
```
Go to Vercel Project Settings → Vars
Add: NEXT_PUBLIC_MAINTENANCE_API_KEY = <your_key>
```

### Step 2: Deploy (1 minute)
```bash
git push origin develop-chatbot-with-api
# or deploy via Vercel dashboard
```

### Step 3: Test (2 minutes)
- Open website
- Click chatbot (yellow button, bottom-right)
- Type: "I need maintenance"
- Complete form

**Total Setup Time: 5 minutes** ⏱️

## 🔧 Technology Stack

- **Framework**: Next.js 15 (React)
- **Language**: TypeScript (100% type-safe)
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: Framer Motion
- **API**: RESTful (JSON)
- **Authentication**: API Key (environment variable)
- **Database**: Via Alazab Maintenance API
- **Deployment**: Vercel

---

## 🎯 Capabilities

### What the Chatbot Can Do

1. **Detect Maintenance Requests**
   - Recognizes keywords in English & Arabic
   - Automatically triggers form

2. **Collect Customer Information**
   - Name
   - Phone number
   - Service type (6 options)
   - Problem description
   - Priority level (3 levels)

3. **Submit Requests**
   - Posts to Alazab Maintenance API
   - Validates all inputs
   - Handles errors gracefully

4. **Provide Confirmations**
   - Shows request number
   - Displays confirmation message
   - Offers tracking capability

5. **Support Bilingual Users**
   - Auto-detects user language
   - Maintains language throughout
   - All responses translated
